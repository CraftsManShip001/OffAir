## (1) 시스템 상태 표시
<br/>
### 위치
프로그램 최상단에 위치합니다
<br/>
### 설명

시스템의 전반적인 상태를 표시해줍니다

방송 상태, 통신 연결 상태, 전원, 타이머 등의 동작 상태를 알려줍니다
<br/>
<br/>
<br/>
## (2) 시스템 INPUT MIXER
<br/>
### 위치

프로그램 좌상단에 위치합니다
<br/>
### 설명

16IN/8OUT 채널 중 16개의 입력을 받아 각 채널별로 선택 제어를 할 수 있습니다

실시간 볼륨조절과 감청, 이퀄라이져, 음소거 기능을 가지고 있습니다
<br/>
<br/>
<br/>
## (3) 시스템 OUTPUT MIXER
<br/>
### 위치

프로그램 가운데에 위치합니다
<br/>
### 설명

8개의 OUT(출력) 채널을 제어 할 수 있습니다

INPUT과 동일한 기능을 가지고 있습니다
<br/>
