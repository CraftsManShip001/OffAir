## 개요
<br/>
외부 소스로서 인터넷에 유튜브로 수신되는 동영상과 USB 카메라 및 BLC-300 TX를 사용하도록 선택 가능한 채널이 2개가 있습니다
<br/>
<br/>
<br/>
## Net1, Net2의 URL 경로 지정
<br/>
인터넷 동영상은 유튜브가 수신 가능합니다

해당 사이트의 URL 주소에서 수신을 확인하여 주세요

1. IP채널 5번, 6번의 환경설정 기어 아이콘을 클릭합니다
2. IP 수신 모드를 선택하여 하단의 URL 주소 입력 창을 확인합니다
3. 기존의 수신 가능한 주소를 선택하여 URL 주소 입력을 완료합니다
<br/>
### 유튜브 새로운 주소 입력
<br/>
1. 화면에서 인터넷상의 유튜브를 접속하고 원하는 동영상을 수신합니다
2. 접속이 되어 수신되면 인터넷 상단의 주소 창을 클릭하여 주소를 복사합니다
3. 다시 메뉴화면으로 이동하여 입력 채널 5번 또는 6번의 URL 주소 창을 연 후 주소 입력 창에 붙여넣기 하여 새로운 주소를 입력합니다
<br/>
<br/>
<br/>
## USB 장치의 영상/음성입력 지정
<br/>
Device탭으로 외부의 USB 방식의 카메라 또는 컨버터 출력 등의 영상을 입력할 수 있습니다
<br/>
### 사용 방법
<br/>
1. USB 카메라 또는 컨버터를 연결 후에 입력채널 5번 또는 6번 채널의 환경 설정을 클릭하여 Device 탭에서 USB 입력모드를 선택합니다
2. 입력 해상도, 오디오 입력 방식을 설정합니다
3. 입력 해상도, 오디오 입력방식을 설정합니다
4. 입력 미리 보기 창에 화면이 보이는 것을 확인하고 설정을 닫습니다
<br/>
### 주의사항
<br/>
입력 장치에 USB장치가 보이지 않는 경우 해당 제품의 드라이버가 설치 되었는지 확인하여 주세요
