## 개요
<br/>
PC에 저장된 여러 가지 콘텐츠 및 외부 입력신호를 미리 순서대로 설정하여 자동 송출하도록 하는 기능입니다
<br/>
<br/>
<br/>
## 순서 송출 기능
<br/>
<img src="/assets/gear/순서 송출/순서 송출.png" alt="참고 이미지" width="80%" height="auto" />
<br/>
<br/>
### 방법
<br/>
1. Play List의 송출 제어 부에서 파일 추가를 누르면 새로운 리스트가 추가됩니다
<br/>
2. 파일열기를 하면 왼도우 검색 창이 열리고 파일 입력 중에서 선택이 가능합니다
<br/>
3. 순서대로 파일이나 입력신호를 선택하면 List가 생성됩니다
<br/>
4. List에서 재생시간을 클릭하여 원하는 재생 시간을 설정합니다
<br/>
5. 송출 제어부의 송출 버턴을 누르면 즉시 송출됩니다
<br/>
### 기능
<br/>
제거 : 삭제하고자 하는 행에서 마우스 오른쪽을 클릭하여 삭제합니다
<br/>
반복 : 반복 재생을 지정합니다
<br/>
내보내기 : 작성된 순서송출 리스트를 모두 저장합니다
<br/>
불러오기 : 저장된 순서송출 리스트를 불러옵니다. 파일 확장자는 .ppf입니다
<br/>
송출 : 작성된 플레이 리스트를 송출합니다. 송출 중에는 프로그램 창 상단에 ‘순서 송출중’ 이라고 표시됩니다
<br/>
중지 : 플레이 리스트를 중지합니다
