## 개요
<br/>
비디오 믹서 기능은 최대 15 채널의 소스영상 신호를 실시간으로 선택이 가능하도록 한 것으로 프리뷰, 프로그램의 직관적인 채널 선택과 다양한 전환 효과 및 전환 시간 선택, Take, Cut 등의 기능을 구현하고 있습니다
<br/>
<br/>
<br/>
## 송출 영상 전환
<br/>
<img src="/assets/gear/VIDEO MIXER/송출 전환1.png" alt="참고 이미지" width="80%" height="auto" />
<img src="/assets/gear/VIDEO MIXER/송출 전환2.png" alt="참고 이미지" width="80%" height="auto" />
<br/>
<br/>
1. 영상 미리 보기 창에서 입력 채널 신호의 영상을 확인합니다
<br/>
2. 프로그램 채널의 선택 : 현재 송출중인 채널을 표시합니다
선택 범위는 외부입력 1~4 채널, DDR1, DDR2, Still, PPT, PC, M1~M4의 15개 채널입니다
<br/>
3. 프리뷰의 선택 : 다음에 송출 대기중인 채널을 표시합니다
<br/>
4. CUT 또는 TAKE 버튼을 누르면 프로그램이 송출합니다
<br/>
<br/>
<br/>
## 전환 효과 기능
<br/>
<img src="/assets/gear/VIDEO MIXER/전환 효과.png" alt="참고 이미지" width="80%" height="auto" />
<br/>
<br/>
전환 효과 설정 창은 Fade, Wipe, Panning, Mix등 약 40개의 다양한 F/X 기능과 사용자 설정의 특수한 전환 효과를 지원합니다
<br/>
1. 수동 전환용 T-BAR로 전환 효과의 위치를 임의로 조작 가능합니다
<br/>
2. 선택된 효과의 표시 및 전환 시간 설정, FADE, FTB 지정을 합니다
전환 시간은 선택버튼 S,M,F 중에서 클릭하여 지정합니다
전환 효과는 M1~M4, PC, PPT 전환시에는 적용되지 않습니다
    1. S : 효과 모드의 동작시간이 slow로 약 2초 진행됩니다
    <br/>
    2. M : 효과 모드의 동작시간이 1초로 진행됩니다
    <br/>
    3. F : 효과 모드의 동작시간이 Fast 약 0.5초로 진행됩니다
    <br/>
    4. Fade : PGM과 PRV 화면이 Take를 누르면 Dissolve로 합성 되면서 전환됩니다
    <br/>
    5. FTB : Fade thru Black 기능으로 전환 시에 PGM과 PRV 화면이 take를 누르면 서서히 검정색이 되고 다시 검정색에서 원래대로 전환됩니다
    <br/>
3. CUT을 누르면 프리뷰 채널이 즉시 프로그램 송출로 실행됩니다
TAKE를 누르면 효과 설정 창에서 지정된 효과가 적용되어 프로그램 송출이 실행됩니다
<br/>
4. 기본 효과 선택 버튼
<br/>

## 사용자 지정 전환 효과 기능
<br/>
<img src="/assets/gear/VIDEO MIXER/사용자 효과.png" alt="참고 이미지" width="80%" height="auto" />
<br/>
<br/>
사용자 전환효과 기능은 다양한 도형이나 문자 등의 개체를 사용자가 작성하여 전환효과에 적용이 가능하도록 한 기능으로 영상 전환 시에 유용한 기능입니다
<br/>
1. 효과 선택창에서 사용자 효과가 등록된 아이콘을 선택합니다
2. 지정하고자 하는 효과를 선택하면 되고 마우스 오른쪽 버튼을 누르면 ‘효과 추가’, ‘편집’, ‘삭제’를 할 수 있습니다
<br/>
### 효과 추가
<br/>
새로운 효과를 작성할 때 누르면 새로운 아이콘이 생성됩니다
<br/>
### 효과 편집
<br/>
수정하고자 하는 효과 아이콘을 선택 후 효과편집이 가능합니다
<br/>
### 효과 삭제
<br/>
삭제할 아이콘 선택 후 마우스 오른쪽 버튼으로 삭제를 선택하면 삭제됩니다
<br/>
### 사용자 효과 편집
<br/>
사용자 효과 편집은 다양한 원하는 효과를 작성하여 사용 가능합니다!
