## 개요
<br/>
프로그램 송출을 매일, 매주, 한번 등으로 시간을 정해서 송출이 가능하도록 하는 기능입니다!

송출 화면은 DDR, 이미지, 입력영상, 음성파일 등을 선택하여 정해진 시간에 자동 송출을 실행합니다
<br/>
<br/>
<br/>
## 예약 시간 지정
<br/>
1. 하단의 기능버튼에서 예약송출을 선택하면 날짜, 시간 설정창이 표시됩니다
2. 하단의 ‘추가’를 선택하면 예약 설정 가능한 설정 list가 표시됩니다
3. 설정 list행의 내용을 차례로 설정합니다
4. 예약 설정이 완료되면 리스트를 저장 ‘내보내기’를 하거나 이전에 설정한 리스트를 ‘불러오기’ 할 수 있습니다
<br/>
### 참고사항
<br/>
- 시간 예약 순서는 설정 순서대로 추가됩니다(송출 순서가 아닙니다!)
- 시간 중복의 경우 노란색으로 경고 표시가 됩니다
- 예약 지정이 설정 되어야 송출됩니다
- 예약송출이 완료, 시간이 지나면 재 송출이 되지 않습니다
