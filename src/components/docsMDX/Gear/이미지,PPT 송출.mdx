## 이미지(Still) 송출
<br/>
1. 동영상 송출방법과 같이 모드 설정 프리셋 창에서 이미지, PPT를 선택하고 파일 열기 아이콘을 눌러 이미지 또는 PPT 파일을 선택합니다
2. 선택된 이미지나 PPT 파일은 갤러리에 등록되고 표시됩니다
이미지, PPT 파일을 두 번 클릭하면 미리 보기 창으로 LOAD되고 이 상태에서 비디오 스위쳐의 프로그램 또는 프리뷰 채널의 버튼으로 송출합니다
3. 동영상 송출 방법 (4), (5)와 동일합니다
<br/>
### 재생 가능한 이미지 파일의 확장자
<br/>
이미지 파일은 jpg, png, bmp 등 갤러리 등록 창에 이미지가 보여지는 이미지들을 송출 가능합니다
<br/>
- .jpg
- .png
- .bmp
- .tiff
<br/>
<br/>

## PPT 파일 송출
<br/>
PPT 파일을 송출하려면 PPT가 설치 되어있어야 합니다

또한 방송 출력으로 송출하기 위해서는 PPT 파일의 버전이 확인되고 PPT 파일 불러오기에서 등록 창에 페이지 미리보기가 표시되어야 합니다

PPT 파일 불러오기를 하면 각각의 페이지는 이미지로 변환되어 각 페이지로 등록되고 송출됩니다
<br/>
1. PPT탭에서 파일열기를 클릭하여 윈도우 탐색 창에서 원하는 PPT 파일을 선택합니다
2. 리스트상에서 송출하고자 하는 PPT 파일을 클릭합니다
3. PPT 미리 보기 창에 선택된 프로젝트의 각가의 페이지가 미리보기에 표시됩니다
<br/>
### PPT 송출
<br/>
1. 송출 페이지의 이미지를 두 번 클릭하여 송출, 다음 페이지 버튼으로 가능합니다
2. 키보드의 엔터 버튼과 스페이스 버튼도 다음페이지 이동 송출 가능합니다
<br/>
### 주의사항
<br/>
PPT의 슬라이드 쇼에 의한 송출(효과, 오디오, 동영상)을 하고자 하는 경우에는 PC 모드로 하여 PC 화면의 슬라이드 쇼를 송출하는 것을 권장합니다
