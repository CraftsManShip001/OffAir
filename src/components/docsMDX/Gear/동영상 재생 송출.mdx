## 개요
<br/>
동영상 재생기능은 PC 폴더에 저장되어 있는 여러 가지 미디어 파일을 재생합니다

PC의 HDD에 저장된 동영상 파일은 DDR로 미디어 채널로 등록할 수 있습니다
<br/>
<br/>
<br/>
## DDR 동영상 재생
<br/>
1. 모드 설정 프리셋 창에서 DDR1 또는 DDR2 탭 폴더를 선택합니다
2. 파일열기 아이콘을 눌러 윈도우 탐색 창에서 재생하고 하는 파일을 선택합니다
3. 갤러리에 설정된 동영상을 두 번 클릭하면 DDR의 미리 보기 창의 DDR 보기에 등록됩니다
4. 비디오 스위쳐 창에서 해당채널의 버튼을 직접 누르거나
5. Take / Cut / T-BAr를 선택하면 채널 전환되어 송출됩니다
<br/>
<br/>
<br/>
## DDR 동영상 재생
<br/>
1. 미리 보기 창에 등록된 동영상은 프로그램 출력 시에 처음/일시정지/재생버튼으로 동영상의 재생위치를 설정할 수 있습니다
2. 설정 기어를 누르면 프로그램 출력이 종료된 후의 동영사의 대기상태를 지정 가능하빈다
<br/>
- 처음 : 프로그램 송출 종료 후 다시 재생될 떄 항상 처음부터 재생
- 일시정지 : 프로그램 출력이 종료되면 정지상태로 대기합니다
- 재생 : 프로그램 출력이 종료되면 재생상태로 대기합니다
- 반복 : 동영상 프로그램 출력 중에 동영상 재생이 반복됩니다
