## 개요
<br/>
Live 영상 송출은 외부 HDMI / YPbPr 단자의 영상 신호를 송출하는 기능입니다
<br/>
<br/>
<br/>
## 영상입력 프로그램 송출
<br/>
1. 외부 영상 미리 보기 창에 입력된 영상 화면이 표시되는지 확인합니다
<br/>
2. 송출 하고자 하는 채널의 화면을 마우스로 두 번 연속 클릭하면 PGM 화면으로 송출됩니다(CUT)
<br/>
3. 현재 송출중인 채널은 프로그램 채널에 빨간색으로 테두리가 표시되며 이전의 채널은 녹색의 테두리로 변경됩니다
<br/>
4. 영상 스위쳐의 프로그램 선택 버튼으로 송출하거나 Take, T-Bar등을 사용하여 채널 변경 시 전환 효과 적용이 되어 송출이 가능합니다
이 경우에는 자동으로 프리뷰 채널과 프로그램 채널로 변경됩니다
