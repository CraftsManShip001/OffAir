## 개요
<br/>
시스템을 사용하기 위해서는 입출력 설정, 각 기능의 설정을 미리 해야합니다

초기에는 기본적인 동작상태로 설정 되어있습니다

사용 환경이나 사용중에 입출력 신호사양 또는 각 기능의 세부 설정이 변경되는 경우에 설정을 변경할 수 있습니다
<br/>
<br/>
<br/>
## 설정 변경
<br/>
<img src="/assets/gear/입력 출력/설정 변경.png" alt="참고 이미지" width="80%" height="auto" />
<br/>
<br/>
설정의 변경은 미리보기 창의 기어 탭과 ‘환경 설정’ 탭에서 설정하는 두 가지가 있으면 각각 설정사항을 편리하게 지정할 수 있습니다

각 창에 있는 설정 기어는 입력, 출력 등의 출력 신호 형식, 해상도, 오디오 형식 등을 지정합니다

Live 입력 창에서는 크로마키 설정도 지원합니다
<br/>
<br/>
<br/>
## Live 입력 설정
<br/>
<img src="/assets/gear/입력 출력/Live.png" alt="참고 이미지" width="80%" height="auto" />
<br/>
<br/>
Live 입력 1,2,3,4의 각각에 설정이 있으며 설정 방법은 동일합니다
<br/>
### 입력 정보 보기
<br/>
입력은 자동으로 신호 인식이 되고, 해상도와 신호표시를 합니다

외부 영상 입력은 1920 * 1080, 1280*720, NTSC, PAL의 해상도 입력이 가능합니다
<br/>
### 크로마키, 루마키 설정
<br/>
해당 채널의 크로마키 설정을 합니다

자세한 설정은 크로마키 설정을 참고합니다
<br/>
### YPbr 입력 비디오 선택
<br/>
<img src="/assets/gear/입력 출력/ypbpr.png" alt="참고 이미지" width="80%" height="auto" />
<br/>
<br/>
입력 3번, 4번은 HDMI와 Component 입력을 겸용으로 하고 있습니다

신호에 따라 선택해주세요

입력 해상도는 자동 설정으로 적용됩니다
<br/>
### Net/USB 1,2 입력 설정
<br/>
Net/USB는 2개의 채널이 있으며 각각에 네트워크 동영상 또는 USB 입력장치, BLC-300 중에서 선택합니다

네트워크의 설정은 인터넷상의 유튜브 동영상 신호의 URL 주소를 입력합니다

USB 입력장치는 외정형 USB 카메라, 마이크 등을 연결하여 영상소스로 사용 가능합니다
<br/>
### USB 카메라 설정
<br/>
<img src="/assets/gear/입력 출력/카메라.png" alt="참고 이미지" width="80%" height="auto" />
<br/>
<br/>
USB 입력 방식의 비디오 컨버터, 카메라, 오디오 마이크, 오디오 믹서기 등을 연결하여 사용할 수 있습니다

1. ‘Video’ 항목에서 설정하고자 하는 장치를 지정합니다
2. USB 카메라에서 사용 가능한 해상도를 설정합니다
카메라마다 지원가능한 해상도가 다르므로 적합한 모드를 선택하여 사용합니다
3. 오디오 디바이스 설정은 USB 카메라에 부착된 마이크나 USB 마이크를 지정합니다
<br/>
### 주의사항
<br/>
USB 카메라를 연결해도 나타나지 않는 경우에는 드라이버를 재설치 하거나 PC를 재부팅 해주세요
<br/>

<img src="/assets/gear/입력 출력/네트워크.png" alt="참고 이미지" width="80%" height="auto" />

<br/>
유튜브 수신의 윈도우 창 상단에 있는 주소를 클릭하여 복사한 후 메뉴의 URL 주소창에 붙여넣기 합니다

## 미리보기 창의 DDR, 이미지, PPT 설정
<br/>
메뉴 하단의 영상 갤러리 탭에서 송출대기 창으로 지정하면 각각의 DDR, 이미지, PPT, PC, M1~M4의 미리보기 창에 표시됩니다
<br/>
### DDR
<br/>
동영상 송출대기 이미지입니다

기어 아이콘을 누르면 송출 설정이 가능합니다

- 송출 후에 미리보기(프리뷰) 창의 대기 상태를 ‘처음’, ‘일시정지’, ‘재생’의 3가지 모드 중에서 선택 지정합니다
- ‘반복’을 선택하면 반복 재생합니다
<br/>
### 이미지
<br/>
모드 설정 프리셋 창의 이미지 중에서 선택된 이미지를 보여줍니다
<br/>
### PPT
<br/>
모드 설정 프리셋 창에서 PPT의 선택된 페이지를 보여줍니다
<br/>
<br/>
<br/>
## 출력 포맷 설정
<br/>
<img src="/assets/gear/입력 출력/포맷.png" alt="참고 이미지" width="80%" height="auto" />
<br/>
<br/>
3개의 출력에는 각각 신호 포맷, 해상도 등을 설정합니다
<br/>
### Output Format
<br/>
출력의 해상도를 설정합니다

표준은 1920 * 1080 59.94i 입니다

다양한 입력 신호가 사용 가능하지만 화질, 화면의 크기가 변경될 수 있으니 동일하게 설정하는 것을 권장합니다
<br/>
### 주의사항
<br/>
입력과 출력 해상도가 다르거나 프레임 주파수가 서로 다른 경우 스크롤 자막의 이동시에 떨림 등이 발생하고 노이즈가 발생할 수 있습니다
<br/>
<br/>
<br/>
## 출력1, 2, 3 설정
<br/>
출력 1,2,3의 각각의 출력은 ‘프로그램’과 ‘입력 1,2,3,4’ 중에서 선택하여 출력 가능합니다

설정 방법은 동일합니다
<br/>
### 프로그램 선택
<br/>
출려보기 창의 상단에 탭을 클릭하면 선택 가능한 소스이름이 표시됩니다

프로그램을 선태갛면 출력표시에 ‘출력 1(프로그램)’으로 표시되고 빨간색이 됩니다
<br/>
### 프로그램(PGM) 선택
<br/>
프로그램 출력은 메뉴상의 스위쳐 기능 및 오디오 믹서기능, 오디오 지연 기능, M1~M4, DDR, PPT 등의 제어 기능이 적용 되는 출력입니다
<br/>
### 입력 1,2,3,4 선택 (라우터 출력)
<br/>
<img src="/assets/gear/입력 출력/라우터 출력.png" alt="참고 이미지" width="80%" height="auto" />
<br/>
<br/>
출력보기 창의 상단에 SUB탭을 클릭하면 선택 가능한 소스이름이 표시됩니다

원하는 것을 선택하면 출력됩니다 (라우터 기능)

동영상, 이미지, 멀티 뷰 등의 송출을 하려면 출력(PGM)으로 선택하여야 합니다

출력 1,2,3의 선택기능으로 다양한 방송 송출이 가능합니다
<br/>
<br/>
<br/>
## 영상 캡처
<br/>
라이브 입력 1,2,3,4 및 출력 1,2,3에는 영상 캡처 기능이 있습니다

상단에 카메라 모양의 아이콘을 클릭하여 1280 * 720의 크기로 이미지 파일로 저장됩니다
