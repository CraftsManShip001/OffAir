## 개요
<br/>
본 제품은 송출 중에 문자를 즉시 작성하여 송출 화면상에 표시하는 기능이 있습니다

또한 문자는 크기, 폰트, 송출 시간, 스크롤 등의 효과를 간편하게 지정이 가능하고 미리보기, 작성된 다문 메시지의 저장이 가능하여 간편한 문자 발생기로 사용할 수 있습니다
<br/>
<br/>
<br/>

### 단문 메시지 작성
<br/>
‘추가’ 버튼을 눌러서 새로운 페이지가 생성되면 문자 입력 창에 문자를 입력합니다
<br/>
### 미리보기
<br/>
미리보기 창에는 각 페이지 별로 미리 보기를 표시합니다
<br/>
- 추가 : 새로운 문자입력 할 때 신규 페이지를 생성합니다
- 삭제 : 단문 미리 보기 창에서 마우스 오른쪽 버튼을 클릭하여 삭제합니다
- 내보내기 : 등록된 단문 문서를 모두 저장합니다
- 불러오기 : 저장된 단문 문서를 업로드 합니다. 파일 확장자는 .ptf 입니다
