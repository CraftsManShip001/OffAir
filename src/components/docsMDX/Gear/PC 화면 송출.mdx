## 개요
<br/>
PC 화면 송출모드는 PC 화면을 그대로 송출하는 기능으로 인터넷, 자료, 사무용 프로그램 등을 송출합니다

다른 송출모드 중에도 PC 송출 On/Off키를 이용하여 편리하게 제어할 수 있습니다!
<br/>
<br/>
<br/>
## PC 화면 송출
<br/>
스위쳐에서 PC를 선택하면 현재 보여지는 PC 화면이 그대로 송출됩니다
<br/>
### PC 화면 송출 하기(Alt + F11)
<br/>
1. PC가 아닌 다른 모드에서 먼저 화면을 최소화합니다
2. 인터넷이나 PC의 파일 등 화면에 송출 하고자 하는 내용을 준비합니다
3. Alt + F11을 눌러서 PC 송출 모드로 하면 현재의 PC 화면이 송출됩니다
<br/>
### PC 화면 송출 종료(Alt + F12)
<br/>
PC 화면 송출 상태에서 단축키 Alt + F12를 누르거나 윈도우 하단에서 최소화된 프로그램 아이콘을 클릭하면 송출이 중단되고 기본 메뉴로 복귀합니다

- Alt + F11 : 송출 프로그램 메뉴를 최소화하고 PC 화면 송출 모드가 됩니다
- Alt + F12 : PC화면 송출모드에서 직전의 모드로 빠져나옵니다
