## 개요
<br/>
Audio Mixer 기능은 4채널의 영상 입력, NET/USB, 외부 오디오, 동영상 등의 PGM용 10채널 스테레오 입력과 출력 1,2,3의 출력 조정기능이 있습니다
<br/>
<br/>
<br/>
## 오디오 믹서 주요 기능
<br/>
### 음소거
<br/>
하단의 스피커 표시 부를 클릭하여 Mute/Mute 해제를 선택합니다

음소거 되면 상단의 빨간색 바가 off되고 해당 채널은 출력이 되지 않습니다
<br/>
### 음량 조정
<br/>
상하의 컨트롤 바를 제어합니다
<br/>
### L/R 밸런스 조정
<br/>
상단의 좌우 조절 레벨 바를 마우스로 클릭하여 이동합니다
<br/>
### 오디오 출력 모드 설정
<br/>
- 믹스 모드 : 비디오 스위쳐의 선택에 관계없이 항상 오디오 채널이 합성되어 출력됩니다
- 연동 모드 : 비디오 스위쳐 채널 선택에 따라 PGM으로 선택된 채널만 오디오가 출력됩니다
- 고정 모드 : 비디오 스위쳐의 선택에 관계없이 항상 선택된 오디오 채널만 출력됩니다. 고정 채널 선택은 상단의 오디오 채널 이름 부분을 누르면 됩니다
<br/>
<br/>
<br/>
## PC 모드 오디오
<br/>
PC 오디오는 윈도우 프로그램에서 PPT, 인터넷 등의 PC 모드시 자체 오디오 재생하는 경우 오디오 조정용으로 사용합니다
<br/>
<br/>
<br/>
## 외부 오디오 출력 지연 설정
<br/>
외부 오디오 출력 XLR 단자와 Phone 출력은 항상 프로그램 영상의 오디오 신호가 출력이 됩니다

현장 설치 시에 영상과 음성 신호의 경로는 결선에 따라서 다를 수 있습니다

사용 장비의 특성으로 시간차가 발생할 수 있습니다

지연 기능을 사용하여 원하는 시간으로 오디오 신호를 지연 설정할 수 있습니다

- 딜레이 시간 : 지연 설정 가능 0~1초 입니다
- 폰 오디오 볼륨 : 후면부의 Phone 출력의 레벨을 제어합니다
<br/>
### 주의사항
<br/>
오디오 지연 출력은 외부 XLR/Phone 오디오 출력단자에만 적용됩니다
