## 개요
<br/>
본 설명서는 기본적인 기능의 구현에서 생성되는 자료 및 데이터를 기준으로 설명합니다

따라서 사용자의 임의 설치프로그램에 대해서는 별도의 설명이 없으며 또한 설치 및 사용을 권장하지 않습니다!

LBS-2000 제품은 출하 시 정상적인 프로그램과 품질시험을 거쳐서 출하되므로 별도의 드라이버나 프로그램 설치가 필요 없습니다
<br/>
<br/>
<br/>
## 시스템의 부팅 및 자동시작
<br/>
제품의 입출력 단자에 신호를 연결하고 전원 버튼을 ON하면 시작화면이 나타나고 자동실행 됩니다
<br/>
<br/>
<br/>
## 프로그램의 시작과 종료
<br/>
방송 시작과 방송 종료 시에 지정된 시간 동안 설치된 이미지를 송출할 수 있습니다

기본으로는 지정된 이미지를 일정시간 지정하는 것으로 세팅 되어있습니다

방송 시작, 방송 종료 이미지는 사용자가 직접 제작하여 등록할 수 있습니다!
<br/>
### 프로그램 시작 시 방송 중비중 이미지 송출
<br/>
프로그램의 시작 시 정해진 시간(~999초) 동안 지정된 이미지를 송출합니다

이미지가 송출 중이라 하더라도 송출 준비중의 메시지 창에서 취소 버튼을 누르면 메뉴 제어가 가능합니다
<br/>
### 프로그램/시스템 종료 시 방송 종료 이미지 송출
<br/>
프로그램 또는 시스템 종료 시에 정해진 시간 동안 지정된 이미지를 송출합니다

메시지 창을 닫거나 취소하면 곧바로 프로그램 또는 시스템이 종료됩니다
<br/>
### 프로그램 종료 후
<br/>
프로그램 종료되면 설정된 입력채널을 출력하는 바이패스 상태를 지정합니다
<br/>
<br/>

## PC 시스템의 환경 설정
<br/>
### 주의사항
<br/>
LBS-2000은 윈도우와 하드웨어의 결합 제품으로 출하 시에 최적의 상태로 세팅됩니다

사용 중에 PC시스템 부분의 설정을 임의로 변경하지 않는 것을 추천합니다
<br/>
### PC SETUP 설정
<br/>
PC의 전원관리 옵션에서 절전모드는 ‘사용 안 함’으로 해주세요

고급설정에서 HDD의 절전모드는 ‘사용 안함’으로 해주세요
<br/>
### 그래픽 메뉴화면 설정
<br/>
출하 시에 1920 * 1080 해상도로 설정되어 있습니다

모니터 변경시에 환결설정을 따로 해주세요
<br/>
### 윈도우 업데이트
<br/>
출하 시에 설치된 OS 프로그램은 최신 버전입니다

임의로 OS를 업데이트 하지 말아주세요
<br/>
### 윈도우 비밀번호 설정
<br/>
PC에 윈도우 관련한 사용자 계정이나 비밀번호 설정은 하지 마세요

부득이 설정 시 반드시 메모 또는 관리를 해주세요

윈도우의 사용자 계정이나 비밀번호 설정에 대한 책임은 사용자에게 있습니다
<br/>
### 메인 프로그램 환경 설정
<br/>
출하 시에 설치된 프로그램의 각종 설정항목은 기본동작상태입니다

프로그램의 환경 설정 창의 비밀번호는 구입시 설치기사에게 문의해주세요
<br/>
### 주의사항
<br/>
메인 프로그램에서 환경설정의 비밀번호는 분실하면 안 됩니다

초기 비밀번호는 12345 입니다
<br/>

## 실행화면
<br/>
### 주의사항
<br/>
메뉴상에서 기어 아이콘을 선택하면 해당 창에 대한 여러 가지의 설정을 할 수 있습니다

메뉴 크기는 전체화면 모드로 해주세요
<br/>
### 메뉴 화면 구성의 설명

1. 메인 메뉴
    1. 기본 메뉴로서 프로그램명, 지원, 등 기본 기능을 표시합니다
2. 영상 입력 미리보기
    1. 영상 외부입력, 선택된 송출대기중의 동영상, 이미지, PPT 미리보기입니다
3. 출력 1,2,3 송출 화면
    1. 메인 프로그램에서 제어에 따라 현재 출력되는 송출화면과 입력 1,2,3,4를 선택하여 출력 가능합니다
4. 비디오 스위쳐
    1. 입력 영상 및 자체 영상 콘텐츠 드르이 송출 채널을 선택할 수 있습니다
    2. 효과 지정, 속도 지정이 가능하고 외부 메트릭스를 제어합니다
5. 효과 지정 및 녹화 설정
    1. 비디오 스위쳐의 효과를 지정합니다
    2. 기본제공 효과 및 사용자 작성효과, T-BAR 등을 제어 설정합니다
    3. 녹화 스트리밍 설정상태를 표시합니다
6. 모드 설정 프리셋(DDR, 오디오, CG, PIP)
    1. DDR, 이미지 등의 지정, 파일 검색, PPT, 멀티 PIP, 오디오 믹서 등의 다양한 기능을 탭으로 제공합니다

## 메인 메뉴 기능
<br/>
### 지원창
<br/>
프로그램 버전, 사용설명, 원격지원 등의 기능을 사용할 수 있습니다

- 원격지원 프로그램 열기
    - AS등의 원격지원을 요청 시 사용합니다
- 메뉴얼 자료
    - 메뉴얼, 퀵 가이드 등의 자료가 있는 폴더를 보여줍니다
    - 원하는 파일을 선택하여 참고해주세요
- 프로그램 버전
    - 프로그램 버전 등을 표시합니다
<br/>
### 녹화, 스트리밍 설정
<br/>
녹화 또는 녹화 중 실시간 확인, 스트리밍의 설정, 관련 정보표시, 시작, 정지 등을 제어합니다
<br/>
### Time
<br/>
현재 시간을 표시합니다
<br/>
### Run
<br/>
프로그램 운용시간을 표시합니다 → 시:분:초로 표시합니다
<br/>
### 종료
<br/>
- 프로그램 종료
    - 방송 송출 프로그램이 종료되고 윈도우 화면으로 이동합니다
    - 프로그램 종료 시 출력화면은 설정된 출력으로 전환됩니다
- 시스템 종료
    - 모든 프로그램이 종료되고 시스템 전원이 꺼집니다
<br/>
### 추가 팁
<br/>
프로그램 종료, 시스템 종료 시 설정 옵션에 따라 바이패스 모드 또는 방송종료 이미지를 일정시간 송출하는 기능을 사용할 수 있습니다
