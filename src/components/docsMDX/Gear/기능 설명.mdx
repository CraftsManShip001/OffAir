## 개요
<br/>
LBS-2000 방송시스템은 HW기반의 제품으로 4개의 HTMI와 2개의 PbPr 입력, IP 수신, USB 카메라 등 다수의 입력과 콘텐츠를 실시간으로 전환하는 스위쳐 기능, 크로마키, CG, XLR단자의 외부 오디오 입출력기능과 지연이 없는 영상 출력, 녹화, 제어 등이 가능한 학교 구내 방송 시스템에 최적화한 방송 시스템입니다
<br/>
<br/>
<br/>
## 특징
<br/>
안정화 보증 - HW 기반 설계

지연 없는 영상출력 - HW Mix 설계

6ch의 영상 입력

3채널의 서로 다른 라우터 출력 기능

3채널 출력의 CG, 단문 메시지 합성 출력 기능

고품질의 XLR단자의 밸런스 오디오 입/출력 기능

파노라마 사진의 가상스튜디오 배경 설정 기능

사용자 작성기능의 다양한 전환효과 기능

프로그램 종료 시 설정된 입력의 출력 기능
