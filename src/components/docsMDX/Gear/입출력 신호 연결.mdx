<br/>

AC 220V 60Hz를 사용합니다
<br/>
### (2) 전원 스위치
<br/>
AC 공급전원을 ON/OFF 합니다
<br/>
### (3) 기본 메뉴 모니터용 HDMI 연결 단자(듀얼 모니터용)
<br/>
### (4) 확장 모니터용 HDMI 연결단자(미사용)
<br/>
### (5) 키보드, 마우스 연결용 USB 단자
<br/>
<br/>
<br/>
## 신호 입출력 단자부
<br/>
1. HDMI 영상신호 입력단자
2. YPbPr 영상신호 입력단자, YPbPr 3,4번 신호는 입력, HDMI 3,4번 채널과 겸용으로 선택합니다
    1. YPbPr 3 ↔ HDMI 3 선택
    2. YPbPr 4 ↔ HDMI 4 선택
3. 출력 1,2,3 단자
4. 외부 Audio XLR 밸런스 입력단자
5. 외부 PGM 출력 Audio XLR 밸런스 출력단자
6. Tally 출력 단자(Open Closed 접점입니다)
7. PGM 출력 오디오 단자(3.5mm)
<br/>
<br/>
<br/>
## 기본 신호 연결
<br/>
### 연결 방법
<br/>
1. AC 220V 전원 플러그를 삽입합니다
2. 키보드
3. 마우스
4. HDMI 출력 케이블을 차례대로 연결합니다
    - 영상신호, 음성신호는 연결도면을 참고하여 연갈합니다
<br/>
<br/>
<br/>
## 영상 입력신호, 출력신호 연결
<br/>
### 연결 방법
<br/>
외부 비디오 입력은 (1)~(6) 채널까지 연결하여 사용 가능합니다

HDMI 3번 ↔ YPbPr3 (5)번 입력, HDMI (4)번 ↔ YPbPr4 (6)번은 서로 겸용이며 선택하여 사용합니다
<br/>
<br/>
<br/>
## USB 방식 영상 입력장치 연결
<br/>
노트북 등의 추가적인 HDMI 신호를 USB 컨버터를 통하여 본체의 USB의 단자에 연결하여 입력 5번 또는 6번 채널로 설정하여 최대 2개까지 확장할 수 있습니다
<br/>
### 노트북의 HDMI → USB 컨버터 연결 방법
<br/>
1. 노트북의 컨버터 사이의 HDMI 케이블을 연결합니다
2. 컨버터 출력의 USB 3.0 케이블을 본체의 후면 USB 단자에 연결합니다
<br/>
### USB 카메라 연결 방법
<br/>
1. USB 카메라(Web Cam)의 연결입니다
2. 카메라의 지원 해상도, 프레임 수를 확인하여 용도에 맞게 설치합니다
<br/>
### 주의사항
<br/>
1. 노트북의 해상도는 1920 * 1080을 권장합니다
<br/>
<br/>
<br/>
## 오디오 신호 입출력 연결
<br/>
### 연결 방법
<br/>
1. 외부 오디오 믹서의 출력을 본체의 오디오 입력 L/R XLR 단자에 연결합니다
2. 본체의 프로그램 오디오 출력을 외부 오디오믹서 입력에 연결합니다
<br/>
### 주의 사항
<br/>
- 밸런스 오디오 입출력 잭의 핀 번호를 확인하여 주세요
밸런스 입력방식이므로 +,-의 극성을 꼭 확인하여 주세요
- 밸런스 ↔ 모노 방식의 상호 연결시 신호이득이 1/2로 저하됩니다
