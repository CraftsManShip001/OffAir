## 개요
<br/>
본 제품은 A/S 또는 프로그램 업데이트, 사용 설명에 관한 문의사항은 인터넷을 통해 원격에서 지원이 가능합니다!
<br/>
<br/>
<br/>
## 원격 지원 접속
<br/>
1. 제품에 인터넷 접속이 가능한 LAN을 연결합니다
<br/>
2. 원격지원을 위하여 Team Viewer를 구입 설치합니다
<br/>
3. 팀 뷰어에 접속이 확인되면 ID와 비밀번호를 A/S 고객지원센터의 서비스 전용전화로 문의하여 전송합니다
<br/>
4. A/S 고객지원 센터의 제품 확인과 점검절차에 따라 주세요
